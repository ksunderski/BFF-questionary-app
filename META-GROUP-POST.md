### **Try It Yourself!**

**Setup Time:** ~5 minutes
**Requirements:** Mac + Chrome + Meta network access

1. **Clone the repo:**
   ```bash
   git clone https://github.com/ksunderski/BFF-questionary-app.git
   cd BFF-questionary-app
   open index.html -a "Google Chrome"
   ```

2. **Get your API key:**
   - Visit: https://wearables-ape.io/settings/api-keys
   - Generate a new key
   - Paste it when the app prompts you

3. **Test with a friend:**
   - Create a questionnaire
   - Generate a friend code
   - Share it with a colleague
   - They redeem the code and fill it out
   - View their answers!
